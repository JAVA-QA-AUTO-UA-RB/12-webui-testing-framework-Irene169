Запуск Cucumber-тестів
Через IDE (IntelliJ IDEA)
1. Відкрити проєкт у IntelliJ IDEA.
2. Відкрити файл раннера, наприклад:
src/test/java/runners/TestRunner.java
3. Натиснути Run біля класу або методу TestRunner.
4. У консолі з’являться результати виконання сценаріїв (Given / When / Then) та статистика, наприклад:
Total tests run: 7, Passes: 7, Failures: 0, Skips: 0

Через Maven
1. Відкрити термінал у корені проєкту.
2. Виконати запуск всіх Cucumber-тестів.
3. Для запуску тестів з певним тегом, наприклад @ui, можна вказати тег у налаштуваннях раннера або в IDE.
4. Після завершення прогона звіт буде збережений у папці:
target/cucumber-reports/
